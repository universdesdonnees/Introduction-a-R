Pour ce projet l'idée est de créer un rapport d'analyse pour les données
d'animation : https://www.kaggle.com/datasets/bangaru5/datasets-related-to-animation

L'objectif est d'explorer les données afin de déterminer par exemple : 

- Top 5 Anime by Popularity
- Average Animation Score by Source
- Top 3 Genres with the Highest Average Animation Score
- Average Popularity and Favorites by Genre Objectiv
- Distribution of Number of Episodes
- Number of anime for the Sunrise' studio 
- Mean number of anime per studio
